## 读题

$n$ 个城市，每个城市有 $b_i$ 单位粮食和每单位价值 $c_i$。若两个城市 $i,j$ 满足 $\max(a_i/a_j, a_j/a_i)$ 是一个质数，则它们可以交易。

目标：在任意两座城市只交易一次的约束下，求总收益的最大值。

## 建模

### 质数比值条件

$a_i, a_j$ 满足条件当且仅当：

1. $\gcd(a_i, a_j) > 1$
2. $a_i / \gcd(a_i, a_j)$ 和 $a_j / \gcd(a_i, a_j)$ 中恰好有一个是质数

### 二分图构造

将所有数字按质因数个数奇偶性分为两组。

- **偶数个质因数**：$1, 4, 6, 8, 9, \ldots$
- **奇数个质因数**：$2, 3, 5, 7, 10, \ldots$

只有奇偶性不同的两个数才能满足"比值是质数"的条件，天然构成二分图。

### 图的样例

以 $2, 3, 6, 9, 10, 15$ 为例，边权为 $b \times c$：

![](/graphs/p4068-graph.svg)

### 费用流

- 源点连所有左节点，容量为 $b_i$，费用为 $0$
- 右节点连汇点，容量为 $b_j$，费用为 $0$
- 二分图中所有合法边，容量为 $\infty$，费用为 $b_i \times c_j + b_j \times c_i$

用 **SPFA 最长路增广** 不断找正收益路径，直到没有正权增广路为止。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

const int MAXN = 305;
const int MAXE = 100000;
const ll INF = 1e18;

struct Edge {
    int to, nxt, cap;
    ll cost;
} e[MAXE];

int head[MAXN], ecnt = 1, S, T;
int cnt[MAXN], n;
ll a[MAXN], b[MAXN], c[MAXN];
ll dist[MAXN], pre[MAXN];
bool inq[MAXN];
queue<int> q;

void add(int u, int v, int cap, ll cost) {
    e[++ecnt] = {v, head[u], cap, cost};
    head[u] = ecnt;
    e[++ecnt] = {u, head[v], 0, -cost};
    head[v] = ecnt;
}

bool spfa() {
    for (int i = 0; i <= T; i++) {
        dist[i] = -INF;
        pre[i] = 0;
        inq[i] = false;
    }
    dist[S] = 0;
    q.push(S);
    inq[S] = true;
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        inq[u] = false;
        for (int i = head[u]; i; i = e[i].nxt) {
            int v = e[i].to;
            if (e[i].cap > 0 && dist[v] < dist[u] + e[i].cost) {
                dist[v] = dist[u] + e[i].cost;
                pre[v] = i;
                if (!inq[v]) {
                    inq[v] = true;
                    q.push(v);
                }
            }
        }
    }
    return dist[T] > 0;
}

int main() {
    scanf("%d", &n);
    S = 0, T = n + 1;
    for (int i = 1; i <= n; i++) {
        scanf("%lld%lld%lld", &a[i], &b[i], &c[i]);
        for (int d = 2; d * d <= a[i]; d++) {
            while (a[i] % d == 0) cnt[i]++, a[i] /= d;
        }
        if (a[i] > 1) cnt[i]++;
    }
    for (int i = 1; i <= n; i++) {
        if (cnt[i] & 1)
            add(S, i, (int)b[i], 0);
        else
            add(i, T, (int)b[i], 0);
    }
    for (int i = 1; i <= n; i++)
        for (int j = i + 1; j <= n; j++) {
            ll g = std::gcd(a[i], a[j]);
            ll xi = a[i] / g, xj = a[j] / g;
            bool ok = (xi == 1 && xj > 1) || (xj == 1 && xi > 1);
            if (ok) {
                ll val = b[i] * c[j] + b[j] * c[i];
                if (cnt[i] & 1) add(i, j, 1e9, val);
                else add(j, i, 1e9, val);
            }
        }
    ll ans = 0;
    while (spfa()) {
        ll flow = 1e9;
        for (int i = pre[T]; i; i = pre[e[i ^ 1].to])
            flow = min(flow, (ll)e[i].cap);
        ans += flow * dist[T];
        for (int i = pre[T]; i; i = pre[e[i ^ 1].to])
            e[i].cap -= flow, e[i ^ 1].cap += flow;
    }
    printf("%lld\n", ans);
    return 0;
}
```

## 调试

实际写代码时遇到了一个坑：增广路的流量上限。

![](/graphs/p4068-debug.svg)

最终发现二分图边的容量不能设为 $\infty$，而应该设为 $\min(b_i, b_j)$。因为每座城市的粮食是有限的，超过这个量的交易没有意义。
