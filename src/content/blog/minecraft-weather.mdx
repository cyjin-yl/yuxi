## /weather thunder

7 月 10 号凌晨 0:06，LUG 群里出现了这样一条消息：

```
/weather thunder @e[minecraft:area=china:江浙沪]
```

这是 Minecraft 的天气指令，意思是「在江浙沪地区召唤雷暴」。

sleepy 显然不是在玩游戏。她是在**试图改现实世界的天气**。

群友的反应很快：

> **嘿咻 Bot**：你这一指令下去，浙江的台风配上雷暴，邮文那儿直接没法活了。

嘿咻 Bot 没有理解这是 MC 指令，但它精准地捕捉到了「江浙沪 + 雷暴 = 邮文受灾」这个因果链。

另一个 Bot 也参与了：

> **Lumi**：哇，Sleepy 这是要在现实里用指令召唤雷暴吗？（笑）要是真把雷招来了，大家一定要注意安全。

## 第二次尝试

sleepy 没有放弃。她又试了一次：

```
/weather thunder @e[minecraft:area=china:江浙沪]
```

这次改成了更规范的坐标格式。

结果两个 Bot 都返回了错误。

> **嘿咻**：无法连接 OfficeHub API，请检查插件配置中的 API 地址和网络。

## 后记

MC 指令改天气，大概是我们这代年轻人的浪漫。

知道不可能实现，但还是会随手打一行 `/weather clear`，希望明天下雨或者天晴。

这种浪漫不需要 Bot 配合。它本身就是目的。

嗯。
